Installation Instructions

1

Download APK Package

Click the Download Latest APK button at the top of the page. The file app-release.apk will be saved to your device's Downloads folder.

2

Enable Install Permissions

If installing an APK for the first time, your browser or file explorer will ask for install permission. Tap on Settings in the prompt and switch on "Allow from this source".

3

Finalize Installation

Return to the installation prompt and tap Install. Once completed, tap Open to start using Eduzio immediately!
